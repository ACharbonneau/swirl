swirl

Statistics with Interactive R Learning (An R Package)

Welcome to swirl!

This is the home for the swirl package for the R statistical programming language. swirl is a platform for learning statistics and R simultaneously and interactively.

This site is designed with three specific groups of people in mind: students, instructors, and contributors. Based on which group you belong to, you will find everything that you need below.

If you are experiencing any problems with swirl, then please go to the bottom of this page for more information.

Students

If you have come here to learn, then you are in the right place. The swirl R package is designed to simultaneously teach users statistics and the R programming language. If you are new to the R programming language, have no fear. We will walk you through each of the steps required to begin using the swirl instructional platform today!

Click here to get swirl now!

Instructors

If you wish to create course content for the swirl instructional platform, then welcome aboard! By following a few simple instructions, you can design an interactive course that will be made accessible to anyone who has the swirl package installed on their computer. You are free to create content on any topic you desire, as long as it involves statistics and R. If you get stuck along the way, we are here to help.

Click here to create your own content now!

Contributors

If you are interested in helping to improve the swirl platform, we welcome your contributions. In the spirit of the R language, the swirl package is totally open source. Although the creators of swirl are passionate about statistics, R, and education, they are not professional software engineers. They recognize that there is plenty of room for improvement. For swirl to continue to mature, it is essential for other passionate and talented individuals to get involved.

If you'd like to help us improve the swirl platform, then feel free to access the code via our swirl GitHub repository. There's a link at the top of this page.

Having Problems? No Problem!

If you are experiencing a problem with swirl, please check the Issues page of our GitHub repository (where we maintain our code) to see if someone else has already posted the same issue.

If no one has done so, then please create a New Issue and include the following items in your post:

  • Name of the course/module within which you are experiencing the problem
  • Brief description of the problem
  • Your operating system and R version number (found by typing R.Version()$version.string in the console)
  • Any screenshots or output that might help us visualize the problem

If you have any other comments or suggestions, or if you'd like to contribute in some other way, then please contact us.
