You're a student!

As you've already learned, swirl is a platform for learning statistics and R simultaneously and interactively. Here, you will find step-by-step instructions on how to get up and running with swirl as soon as possible. Since the swirl package is still young at heart, the process of downloading it is slightly more involved right now than it will be in the future.

Step 1: Get R and RStudio

In order to run swirl, you must have R installed on your computer. In addition to R, it highly recommended that you install RStudio, which will make your experience with R much more enjoyable.

If you need to install R, you can do so here.

If you need help installing R, check out one of the following videos (courtesy of Roger Peng at Johns Hopkins Biostatistics):

  • Installing R on Windows
  • Installing R on Mac

If you need to install RStudio, you can do so here. You probably want the download located under the heading Recommended For Your System.

Step 3: Download devtools package
